# 言語設定
日本語化する。  
まずは、Extensionsで"Japanese Language Pack for Visual Studio"をインストールする  
次に、View -> CommandPalette -> "Configure language"と入力  
JSONファイルを"locale":"ja"に変更  
[参考 Qiita](https://qiita.com/ayatokura/items/1efb36d54625dc307ab2)
