# node.js

## プログラムの実行方法
node.jsをインストールしておく  
VS Code標準でサポートしているので、ターミナルを開いて  
node app.js  
で実行できる。

## デバッグ方法
### デバッグ構成を設定する  
Ctrl + Shift + D でデバッグ画面に遷移  
Add Configurationを選択
![デバッグ構成の追加](img/add_configuration.png)  
すると、launch.jsファイルが作成される  
![launch.js](img/launchjs.png)  
  
最後にF5でデバッグ開始！
