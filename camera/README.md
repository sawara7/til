## デジカメ入手  
・レンタルもありかも  
郵送で3泊4日レンタルのものがある。カメラ単体で7,8千円する。  
レンズも揃えるとかなり高額になる。  
東京なら直接受け渡しする窓口がある店舗があるので、そちらを使えば当日、一泊二日で安く済ませることができる。
